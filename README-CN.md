## 安装
通过composer安装
```
composer require write-doc/write-doc
```

## 使用
```
php builder.php demo
```

## 特性
* 使用markdown语法编写文档,API文档编写更简易
* 自动生成链接目录
* 自动生成链接跳转的锚点
* 使用`j`和`k`快捷键在api间快速跳转
